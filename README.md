# API Key UI

API Key 前端管理插件，适用于 FastAPI Best Architecture 的 Vben Admin Antd 工程。

## 功能

- API Key 列表分页与名称搜索
- 新增 / 编辑 API Key
- 快捷启用 / 停用
- 掩码展示与复制令牌
- 有效期设置与快捷预设

## 使用说明

将本仓库中的所有文件拷贝到前端工程插件目录中，例如：

`apps/web-antdv-next/src/plugins/api_key_ui`

该插件依赖后端 `api_key` 插件提供接口能力。
